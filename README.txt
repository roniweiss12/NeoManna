This repository describes all the CoExpNetViz graphs created so far. 
In the Matrices directory are the originial metrices (already normalized but unfiltered) used in each graph (by author's name) and the filtered ones.
In the Networks_output directory are the output files created in from each dataset, divided by organism and by the type of baits file used.
The file "datasets_for_coExp" describes the charactaeristcs of each dataset used in this project.
In the "blast2GO" directory are the output from the software as gff files and the input which is a list of all the genes accessions.
